# ASCPL

A simple Programming Language inspired by TypeScript & C.

## Getting Started

1. First, clone the repo.
```bash
git clone https://github.com/itzkbp/ASCPL.git
```
2. Then, cd into ASCPL directory
```bash
cd ASCPL
```
3. Use the makefile
```bash
make
```
